# Javascript #1 변수(variable)

## 변수사용법
- 변수 : 데이터를 담는 메모리

1. 보관함 확보 : 선언 (Declaration)
Ex) let country;

2. 보관함에 데이터 저장 : 할당(assignment)
Ex) country = Korea;

3. 선언과 할당 동시
Ex) let age = 25;

- 크롬 개발도구 console에서 작업 가능 (shortcuts : command+option+i)


# Javascript #2 타입(type)

 - 객체 : Form 설문지 같은 형식
